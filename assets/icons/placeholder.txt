PNG icon files needed:
- icon16.png (16x16 pixels)
- icon32.png (32x32 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

You can:
1. Convert the provided icon.svg to PNG using online tools
2. Create simple colored squares as placeholders
3. Use the extension without icons (remove icon references from manifest.json)

The extension will work without icons, they're only for visual appearance.
